
# Commute Checker

## Description

A beginner Python program that determines whether commuting is possible based on distance, weather conditions, and available transportation. The program was created as a practice exercise while learning Python and conditional logic.

## Concepts Practiced

* Variables
* Boolean values
* Conditional statements (`if`, `elif`, `else`)
* Comparison operators
* Logical operators (`and`, `or`)
* Decision-making based on multiple conditions

## Learning Source

This project is of a programming exercise inspired by the FreeCodeCamp Python curriculum with extra things that I added.

## Skills Practiced

* Problem-solving
* Translating requirements into code
* Reading and implementing user stories
* Writing clear conditional logic
