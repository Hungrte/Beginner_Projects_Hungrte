# Caesar Cipher

## Description

A beginner Python implementation of the Caesar Cipher that can encrypt and decrypt text by shifting letters of the alphabet. This project was created while learning Python fundamentals and demonstrates the use of functions, string manipulation, slicing, and translation tables.

## Features

* Encrypts text using a configurable shift value
* Decrypts encrypted text
* Supports both uppercase and lowercase letters
* Validates the shift value
* Uses Python's `str.maketrans()` and `translate()` methods for efficient character substitution

## Concepts Practiced

* Functions
* Parameters and return values
* Conditional statements
* Boolean logic
* String methods
* String slicing
* `str.maketrans()`
* `str.translate()`
* Input validation
