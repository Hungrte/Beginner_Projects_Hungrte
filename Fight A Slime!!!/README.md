# CS50 Scratch Turn-Based Battle Game

## Description

A simple turn-based game that features three elemental spells, a status effect, and a game-ending mechanic. The game is played entirely through keyboard input.

## How to Play

1. Click the **Green Flag** to start the game.
2. Wait a few seconds for the dialogue to begin.
3. When prompted, type one of the following:

   * **1** — Fireball
   * **2** — Stun
   * **3** — Water Splash
4. Press **Enter** to confirm your choice.
5. After your turn, the slime will attack. Turns alternate between you and the slime until one of you is defeated.

## Features

* Turn-based combat
* Three elemental spells
* Stun status effect
* Win/Loss game-ending mechanic

* Scratch
