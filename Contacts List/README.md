# Contact List (Python)

A simple command-line contact list built in Python.

## Features

- Add a contact
- Search for an existing contact
- List all saved contacts
- Input validation for menu options and contact numbers

## Concepts Practiced

- Functions
- Conditional statements
- Loops
- Lists
- String methods (`isdigit()`, `startswith()`, `len()`)
- Input validation
- Basic program structure

## Possible Future Improvements

- Store contact names using dictionaries
- Save contacts to a file
- Edit and delete contacts
- Implement partial search/autocomplete
- Create a graphical user interface
